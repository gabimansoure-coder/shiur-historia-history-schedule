# שיעור היסטוריה - מתכונת מלחמה (אתר מקומי)

קבצים:

- `index.html` — דף הראשי: בורר כיתה ודינמיית הצגת הקישורים (הטבלה הוסרה)
- `styles.css` — עיצובים עבור RTL, בורר ותצוגת תוצאות

הוראות מהרצה:

1. פתח את הקובץ `index.html` בדפדפן (לחיצה כפולה או גרירה לדפדפן).
2. בממשק: בחר את הכיתה מהרשימה ולחץ על `הצג` — יופיעו רק הקישורים המתאימים.
3. לא נדרש שרת; זהו אתר סטטי.

שינויים עיקריים:

- הטבלה הוסרה — הנתונים נטענים מתוך מערך JavaScript פנימי והבורר נבנה דינמית.
- אם תרצה, אשמור את הנתונים ב-JSON חיצוני או שאוסיף כפתור "העתק קישור" ליד כל פריט.

האם ברצונך שאאוסיף כפתור "העתק קישור" לרשימת התוצאות או שאייצא את הנתונים כקובץ JSON חיצוני?
